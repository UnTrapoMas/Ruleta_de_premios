# Ruleta de Premios

Una ruleta interactiva para promociones y sorteos.

## Características

- Ruleta visual con animaciones suaves
- Premios con diferentes probabilidades
- Sistema de reclamo de premios vía Instagram
- Diseño responsive y moderno
